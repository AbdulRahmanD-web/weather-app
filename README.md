
Приложение погоды

Адаптивное приложение для прогноза погоды, созданное с использованием HTML, CSS и JavaScript с помощью WeatherAPI.

Функции

Текущая погода
Прогноз на 7 дней
Поиск городов
Обработка ошибок
Проверка запроса
Адаптивный дизайн
